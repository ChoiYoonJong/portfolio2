Project

이세돌 팬사이트 (Isedol Fansite)
JSP/Servlet 기반 회원·게시판 CRUD 웹 애플리케이션 — 버추얼 아이돌 그룹 소개 & 커뮤니티 사이트
역할 백엔드 설계 및 구현
패턴 MVC2 (Controller–Service–DAO–VO)
DB Oracle · JDBC (쿼리 .properties 외부화)
인증 세션 기반 로그인 + 비밀번호 암호화 필터

6인조 버추얼 아이돌 그룹을 소재로 한 팬 커뮤니티 웹사이트입니다. 회원가입/로그인 등 계정 기능과 공지사항 게시판 CRUD, 멤버별 소개 페이지로 구성했고, 요청 처리 전 구간에 인코딩 필터를, 계정 관련 요청에는 암호화 필터를 별도로 적용해 공통 관심사를 컨트롤러 밖으로 분리했습니다.

  • 회원가입 / 로그인 / 로그아웃 / 마이페이지 / 비밀번호 변경 / 회원 탈퇴
  • 공지사항 게시판 CRUD (목록·상세·등록·수정·삭제)
  • 멤버 6인 개별 소개 페이지 — 서블릿 라우팅으로 분리
  • EncodingFilter로 전 구간 UTF-8 인코딩 일괄 처리
  • EncryptFilter로 로그인·가입·비밀번호 변경 시 암호화 적용
  • SQL 쿼리를 .properties 파일로 외부화해 코드와 분리
